# Abygail Charles — Personal Website

A minimal, professional personal site for Abygail Charles — tutor and recent high school graduate based in East Trinidad & Tobago.

## Stack

- [Next.js 15](https://nextjs.org/) (App Router)
- [Tailwind CSS 4](https://tailwindcss.com/)
- Deployed on [Vercel](https://vercel.com/)

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the repository
4. Click **Deploy** — no extra config needed

Your site will be live at `your-project.vercel.app`.

## Customisation

- **Clubs**: Edit `src/components/Clubs.tsx` with your real extracurricular list
- **Projects**: Edit `src/components/Projects.tsx` with your actual projects
- **Colours**: Defined in `src/app/globals.css` (light pink, butter yellow, baby teal)
